# Test-tecnique-COLBR

### Installation


    composer install

### 

    cd test
    test/.env
    
 ### Configure .env 
    
    DB_CONNECTION=   //the db that you're using(mysql, pgsql...)
    DB_HOST=localhost 
    DB_PORT=    //db port
    DB_DATABASE=   //db name
    DB_USERNAME=   //your db username or root...
    DB_PASSWORD=   //db password if any
    
    
## Start Project

    cd test
    php artisan serve
